## 新增 Agent 的流程（設計目標）

新增一個 Agent 只需要：

1. 在資料庫新增一筆 Agent 紀錄（名稱、職責描述、所屬團隊）
2. 在 Notion 建立該 Agent 的規則庫頁面
3. 實作該 Agent 的 Prompt 與執行邏輯
4. CEO 下次被呼叫時，自動感知新成員存在

不需要修改 CEO 的核心邏輯，也不需要修改 Discord Bot 框架。
